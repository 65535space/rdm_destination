_calculateDistanceメソッド
このメソッドは、2つの地点（startとend）間の直線距離を計算します。
ここで使われているのは「ハバースィンの公式」で、
これは地球上の2点間の大円距離（最短距離）を計算するために用いられます。
地球は球体に近い形状をしているため、この公式は地球表面での距離計算に適しています。
